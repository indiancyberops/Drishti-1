## ABOUT TOOL :

Drishti is an information collection tool (OSINT) which aims to carry out Basic Information. It provides various modules that allow efficient searches. Drishti does not require an API key or login ID.


## AVAILABLE ON :

* Termux
* Windows
* Linux

* Kali Linux
